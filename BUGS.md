# x-logview Bug 列表

## 待修复

（暂无）

## 已修复

### BUG-001: 左侧文件列表与标签页不同步
- **修复时间**: 2026-05-23
- **修复方案**: 修改 Sidebar 组件接收 files prop，从 App.tsx 的 tabs 状态获取文件列表

### BUG-002: 关闭窗口再打开后不显示之前的文件
- **修复时间**: 2026-05-23
- **原因**: 窗口关闭时应用退出，localStorage 可能未保存
- **修复方案**: 打开文件时立即保存到 localStorage

### BUG-003: Sidebar无限重试加载sessions
- **修复时间**: 2026-05-23
- **原因**: Sidebar 重试逻辑没有上限
- **修复方案**: 添加最大重试次数限制（3次）

### BUG-004: Electron启动后提示"正在连接后端服务..."
- **修复时间**: 2026-05-23
- **原因**: 前端在后端启动完成前就开始请求 API
- **修复方案**: 增加前端连接延迟到3秒，Sidebar加载延迟到4秒

### BUG-005: 打开文件后一直显示loading
- **修复时间**: 2026-05-23
- **原因**: 后端 Send 函数没有设置消息ID，前端无法匹配响应
- **修复方案**: 添加 SendResponse 函数，所有处理器使用 SendResponse

### BUG-006: 关闭程序再打开，新建未保存文件没有打开
- **修复时间**: 2026-05-23
- **原因**: 新建文件没有保存到持久化目录
- **修复方案**: 新建文件保存到用户数据目录 ~/.x-logview/temp/

### BUG-007: 新建文件后显示Loading...
- **修复时间**: 2026-05-23
- **原因**: 新建文件路径格式变化，检测逻辑不匹配
- **修复方案**: 更新检测逻辑，使用 file.path.includes('untitled-')

### BUG-008: 新建文件内容保存到/tmp目录会丢失
- **修复时间**: 2026-05-23
- **原因**: 临时文件保存到/tmp目录，电脑重启后会丢失
- **修复方案**: 改为保存到用户数据目录 ~/.x-logview/temp/

### BUG-009: CSP安全警告（Insecure Content-Security-Policy）
- **修复时间**: 2026-05-23
- **原因**: CSP策略中使用了unsafe-eval
- **修复方案**: 移除unsafe-eval，更新CSP策略

### BUG-010: WebSocket连接localhost失败
- **修复时间**: 2026-05-23
- **原因**: Electron主进程的http模块无法正确解析localhost
- **修复方案**: 将localhost改为127.0.0.1

### BUG-011: 编码检测错误（isGBK/isBig5/isShiftJIS）
- **修复时间**: 2026-05-23
- **原因**: 编码检测函数实现不正确
- **修复方案**: 重新实现编码检测逻辑

### BUG-012: 缓冲区大小计算错误
- **修复时间**: 2026-05-23
- **原因**: Read函数中endOffset-startOffset可能为负数
- **修复方案**: 添加边界检查，当startOffset >= endOffset时返回空结果
